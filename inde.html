Project Tracking Software

Project tracking software is a web application for organizations and projects. It can be used different areas in any project development process or event organizations.
The application will include different Boards that users can access into. Boards will represent one project that is lead by a Superuser. Each Board will have a team under working them.
SuperUser of a Board will be able to create different Lists inside a Board, which represents the sub-tasks inside a project. The SuperUser also will be able to add participants to the Team and to the Board. Additionally, the software will able to create different cards inside the lists, which are the small tasks need to be completed. Each card will be assigned to different users, and will have start dates and deadlines. Users can access to the cards that are assigned to them and complete the tasks, also can add some comments to the each task card. SuperUsers will be able to assign tasks to each user, and also will be able to assign list leading role to List Leader, which represents the project leaders in a team
